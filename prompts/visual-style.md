# Visual Style Prompt

Use this base prompt for all images:

```text
Original high-quality 3D animated family movie style, anthropomorphic fruit-headed contestants with expressive fruit faces, no human faces, cinematic lighting, colorful reality TV villa, soft rounded shapes, glossy render, vertical TikTok composition, comedy reality show energy, no copyrighted characters, no existing studio style, no existing celebrity likeness.
```

Negative prompt:

```text
No human face, no realistic human head, no body-shaming of real humans, no humiliating human pose, no grotesque human exaggeration, no real celebrity, no copyrighted character, no brand logo, no medical before-after ad style.
```
