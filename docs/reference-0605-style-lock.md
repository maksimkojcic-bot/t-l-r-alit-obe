## What The User Wants

This is the quality and style target:

- vertical TikTok video, 1080x1920;
- glossy 3D cartoon reality show;
- luxury tropical villa, pool, palm trees, neon name signs;
- anthropomorphic fruit heads with expressive eyes and mouths;
- human-like cartoon bodies with very large round bellies;
- clean, colorful, glamorous, saturated visuals;
- confession-room shots with character names behind them;
- intro cards and episode cards like reality TV;
- fast pacing with no dead air;
- actual animated video clips, not a still image with shake;
- lip-sync or at least clear mouth movement when characters speak;
- white bold subtitles with black stroke only, no black subtitle rectangle;
- CapCut Pro can be used for final auto-captions if needed.

## What To Avoid

- dirty or horror-looking characters;
- gritty, grimy, disgusting backgrounds;
- static image-only scenes as final output;
- black subtitle boxes;
- real human faces;
- jokes aimed at real people or protected groups;
- copying logos, TV channel branding, or platform UI from references.

## Production Role Split

ChatGPT or image model:

- character key art;
- villa/background shots;
- clean reference images for each scene.

Gemini:

- stronger prompts;
- continuity notes;
- shot planning.

Kling, Veo, or another image-to-video model:

- character movement;
- speaking poses;
- walk-ins;
- reaction shots;
- villa motion;
- 4-6 second scene clips.

ElevenLabs:

- final voices;
- one voice per recurring character.

Lip-sync tool:

- mouth movement for talking shots if image-to-video does not handle it well.

Remotion:

- final assembly;
- timing;
- transitions;
- sound effects;
- music;
- optional captions;
- final MP4 export.

CapCut Pro:

- optional final subtitle polish;
- optional TikTok-native effects if the user wants manual finishing.
