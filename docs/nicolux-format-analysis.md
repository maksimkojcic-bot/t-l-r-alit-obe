## Ce qui marche dans le format

### 1. POV immediat

Le contexte est compris en 1 seconde :

- "Chez le medecin"
- "Pov : Quand je fais du velo"
- "Pov : Le drive"
- "Pov : A la salle de sport"

Adaptation Objectif 70 :

- "POV : premier matin dans la villa"
- "POV : la balance annonce un truc chelou"
- "POV : le coach a trouve le tiroir a biscuits"
- "POV : atelier sushi mais t'as pas le droit au riz"

### 2. Personnage central tres expressif

Le comique vient du visage, du souffle, de la posture et de la mauvaise foi.

Regle pour nos videos IA :

- plans serres sur le visage ;
- reactions trop longues juste avant la chute ;
- zoom lent sur les yeux quand le personnage ment ;
- silence de 0,3 seconde avant la punchline.

### 3. Langage oral et direct

Le registre est populaire, tres parle, parfois grossier. On doit viser cette energie sans copier les phrases.

Lexique autorise :

- "frere", "chef", "wesh", "mais t'es malade", "arrete tes conneries", "c'est quoi ce bordel", "j'en peux plus", "il abuse", "c'est mort", "wallah" seulement si coherent avec le personnage.

Gros mots utilisables avec moderation :

- "putain", "merde", "bordel", "fait chier", "connerie".

Pour la monetisation :

- garder les gros mots surtout dans l'audio ;
- sous-titres legerement adoucis si besoin : "p*tain", "m*rde" ;
- jamais d'insulte degradee contre le corps ou une categorie de personnes.

### 4. Mauvaise foi comme moteur

Presque chaque sketch fonctionne avec une excuse impossible :

- "Je fais pas du sport, je surveille le velo."
- "C'est pas manger, c'est analyser."
- "Si je renifle le donut, les calories rentrent pas."

Adaptation Objectif 70 :

- la tentation devient un debat juridique ;
- le candidat invente une regle qui l'arrange ;
- le coach prend la situation trop au serieux ;
- le presentateur annonce une sanction absurde.

### 5. Autorite absurde

Medecin, coach, police, drive, restaurant : une autorite ou un cadre social normal devient ridicule.

Adaptation :

- le coach parle comme un militaire ;
- la balance parle comme un juge ;
- le frigo devient un coffre-fort ;
- le buffet a un vigile ;
- le sushi chef annonce des regles de guerre.

## Formule d'episode 60s

0-3s : Titre POV tres lisible.  
3-8s : Situation impossible.  
8-18s : Le personnage essaie de rester normal.  
18-32s : Mauvaise foi / excuse / mensonge.  
32-45s : Autorite absurde intervient.  
45-55s : Chute grossiere mais pas haineuse.  
55-60s : Cliffhanger ou reaction muette.

## Signature Objectif 70

On ne fait pas "le meme Nicolux". On fabrique une version originale :

- meme efficacite POV ;
- meme energie orale ;
- meme gout pour la mauvaise foi ;
- plus de structure de serie ;
- personnages recurrents ;
- cliffhangers ;
- attention a la monetisation.
